# ربات آگهی تلگرام

یک ربات آگهی پیشرفته برای تلگرام با قابلیت‌های متنوع

## ویژگی‌ها

### سیستم آگهی
- ثبت آگهی با عنوان، توضیحات، قیمت، دسته‌بندی و تصاویر
- سیستم هشتگ هوشمند برای هر دسته‌بندی
- محدودیت تعداد آگهی برای هر کاربر
- اعتبار زمانی برای آگهی‌ها

### سیستم جستجو
- جستجو بر اساس متن، هشتگ و دسته‌بندی
- فیلتر بر اساس قیمت و تاریخ
- مرتب‌سازی نتایج
- نمایش نتایج به صورت صفحه‌بندی شده

### سیستم امتیازدهی و نظرات
- امتیازدهی به آگهی‌ها (۱ تا ۵ ستاره)
- ثبت نظر برای آگهی‌ها
- گزارش آگهی‌های نامناسب
- نمایش میانگین امتیازات

### سیستم اعلان‌ها
- اعلان آگهی‌های جدید در دسته‌بندی‌های مورد علاقه
- اعلان پیام‌های جدید
- اعلان انقضای اشتراک
- اعلان تایید یا رد آگهی

### سیستم پرداخت
- پرداخت آنلاین با درگاه پرداخت
- پرداخت کارت به کارت
- تاریخچه پرداخت‌ها
- تایید خودکار پرداخت

### پنل ادمین
- مدیریت کاربران
- مدیریت آگهی‌ها
- مدیریت نظرات و گزارش‌ها
- ارسال پیام همگانی
- مشاهده آمار و گزارش‌ها

## نصب و راه‌اندازی

### پیش‌نیازها
- Python 3.8 یا بالاتر
- MongoDB
- Redis (اختیاری)
- توکن ربات تلگرام
- درگاه پرداخت (اختیاری)

### نصب
1. کلون کردن مخزن:
```bash
git clone https://github.com/yourusername/telegram-ad-bot.git
cd telegram-ad-bot
```

2. نصب وابستگی‌ها:
```bash
pip install -r requirements.txt
```

3. تنظیم متغیرهای محیطی:
```bash
cp .env.example .env
```
و سپس مقادیر را در فایل `.env` تنظیم کنید.

4. اجرای ربات:
```bash
python bot/main.py
```

## تنظیمات

### فایل config.py
- تنظیمات قیمت اشتراک
- تنظیمات درگاه پرداخت
- تنظیمات اعلان‌ها
- تنظیمات رابط کاربری
- تنظیمات جستجو
- تنظیمات امتیازدهی
- تنظیمات آگهی
- تنظیمات پیام

### فایل .env
```
BOT_TOKEN=your_bot_token
ADMIN_ID=your_admin_id
MONGODB_URI=mongodb://localhost:27017
DB_NAME=telegram_bot
PAYMENT_GATEWAY=zarinpal
MERCHANT_ID=your_merchant_id
PAYMENT_CALLBACK_URL=your_callback_url
BANK_CARD=your_bank_card
```
